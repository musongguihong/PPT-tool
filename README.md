# PPT-tool
# ReadMe
### 项目介绍
这个项目是一个用Python编写的脚本，旨在将PowerPoint演示文稿中的文字内容批量导出到Word文档中。通过这个工具，用户可以更方便地将演示文稿中的文字提取出来，以便于编辑、共享或者进一步处理。
